# SOP (Standard Operating Procedures)

Placeholder folder for site/organization Standard Operating Procedures.

No documents have been added yet. This folder holds a place in the corpus structure until the organization's own SOPs (or other permitted internal procedure documents) are available to ingest.

## Adding documents

- Only add SOPs you have the right to use — typically internal documents owned by the organization deploying SafeFusion AI.
- Record the SOP identifier/title, owning department, version, and effective date for each document added (see corpus [README.md](../README.md#provenance-tracking)).

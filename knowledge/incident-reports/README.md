# Incident Reports

Placeholder folder for historical incident/accident reports used as RAG source material.

No documents have been added yet. This folder holds a place in the corpus structure until legitimately sourced incident reports (internal investigation reports, or publicly available regulatory incident reports) are available.

## Adding documents

- Only add incident reports you have the right to use. Internal incident reports may contain sensitive or personal information — apply appropriate redaction/anonymization before ingestion.
- Record the incident date, site/facility, report source, and retrieval/ingestion date for each document added (see corpus [README.md](../README.md#provenance-tracking)).
